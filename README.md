# Actions-ImmortalWrt-360T7

![immortalwrt](logo.png)

## 🤖关于此脚本与固件
- 脚本内已中文备注，方便理解与自定义  

- 两种固件源码版本，分别来自于hanwckf & padavanonly  

- .config文件自定义可参考[OpenWrt插件名称对照](https://www.right.com.cn/forum/thread-344825-1-1.html) 

- 固件默认开启：ZRAM、MWAN3、ACC 

- 固件默认网关：192.168.10.1
